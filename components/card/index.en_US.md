## API

| Property | Description                        | Type         | Default |
| -------- | ---------------------------------- | ------------ | ------- |
| title    | Card title                         | String, slot | -       |
| icon     | Icon for the card title            | String       | -       |
| bordered | Whether the card displays a border | Boolean      | true    |
| extra    | Card title extension               | slot         | -       |
