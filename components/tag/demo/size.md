<cn>
#### 尺寸和形状
通过 `size` 可控尺寸 ,
</cn>

```vue
<template>
  <Space>
    <Tag>标签1</Tag>
    <Tag closeable size="middle">标签2</Tag>
    <Tag closeable size="large">标签3</Tag>
    <br/>
    <br/>
    <Tag closeable shape="circle">标签1</Tag>
    <Tag closeable shape="circle" size="middle">标签2</Tag>
    <Tag closeable shape="circle" size="large">标签3</Tag>
  </Space>
</template>
```
