## Tag API
| 属性      | 说明                                       | 类型     | 默认值 |
| --------- | ------------------------------------------ | -------- | ------ |
| closeable | 是否显示关闭按钮                           | Boolean  | false  |
| color     | 标签的颜色                                 | String   | -      |
| iconn     | 标签的图标                                 | String   | -      |
| close     | 关闭标签的回调事件                         | Function | -      |
| size      | 按钮尺寸,可选值 `small`、`large`，默认不选 | String   | -      |
