## Loading API

| Property | Description              | Type              | Default |
| -------- | ------------------------ | ----------------- | ------- |
| start    | Start loading            | Function          | -       |
| finish   | Finish loading           | Function          | -       |
| error    | Loading error            | Function          | -       |
| update   | Manually update progress | Function(percent) | -       |
| destroy  | vue's $.destroy()        | Function          | -       |
