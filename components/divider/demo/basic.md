### 垂直分割线
使用 type="vertical" 设置为行内的垂直分割线。
</cn>
<en>
### Vertical Divider
Use type="vertical" to set an inline vertical divider.
</en>

```vue
<template>
  <div>
    Text
    <Divider type="vertical" />
    <a href="#">Link</a>
    <Divider type="vertical" />
    <a href="#">Link</a>
  </div>
</template>
```
