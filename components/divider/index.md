# Divider 分割线

区隔内容的分割线。

## 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。

<code src="./demo/withText.vue">带文字的分割线</code>

## API

| 参数        | 说明                                  | 类型    | 默认值     |
| ----------- | ------------------------------------- | ------- | ---------- |
| text        | 分割线文字                            | String  | -          |
| dashed      | 是否虚线                              | Boolean | false      |
| orientation | 分割线标题的位置：left right          | String  | center     |
| type        | 水平还是垂直类型: horizontal vertical | String  | horizontal |
